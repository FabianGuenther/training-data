Trainingsplan 11.05. - 24.05.2026 (14 Tage)

Erweiterte Recovery-Phase mit sanftem Aufbau zur Trail-Vorbereitung

1) Lagebewertung

Du befindest dich in der erweiterten Recovery-Phase nach der erfolgreichen Doppelbelastung (Wien Marathon + Eschborn Frankfurt). TSB -1.6 und ACWR 1.21 zeigen optimale Erholung. Phase "Base" (medium confidence) mit TID-Alert "shifting" ist normal nach Marathon-Recovery. Zugspitz Ultratrail in 39 Tagen (19. Juni) - strukturierte Trail-Trainingsphase startet ab 19. Mai gemäß Makroplan.

2) Tagesplan

Woche 1: 11.05. - 17.05.2026

Montag 11.05. - Ruhetag (fix) Art: Komplette Ruhe

Dienstag 12.05. - Easy Run mit progressivem Aufbau Art: Easy Laufen Dauer: 50-60 min Herzfrequenz: Z1-Z2 (135-150 bpm) Pace: 5:00-5:30/km Fokus: Progressiver Aufbau nach Recovery-Phase, längere Laufeinheit

Mittwoch 13.05. - Easy Radfahren Art: Easy Radfahren Dauer: 75-90 min Herzfrequenz: Z1-Z2 (< 150 bpm) Fokus: Cross-Training, Ausdauerbasis festigen

Donnerstag 14.05. - Easy Run + Krafttraining Art: Easy Laufen + Krafttraining Dauer: 45 min Lauf + 45-60 min Kraft Herzfrequenz: Z1-Z2 (135-150 bpm) Pace: 5:00-5:30/km Fokus: Laufspezifisches Krafttraining: Hüfte, Core, Stabilität

Freitag 15.05. - Easy Radfahren Art: Easy Radfahren Dauer: 60-75 min Herzfrequenz: Z1-Z2 (< 150 bpm) Fokus: Aktive Recovery zwischen Laufeinheiten

Samstag 16.05. - Long Run 2 Art: Long Run Dauer: 90-105 min Herzfrequenz: Z1-Z2 (135-150 bpm) Pace: 5:00-5:30/km Fokus: Längste Laufeinheit der Woche, Vorbereitung auf Trail-Phase

Sonntag 17.05. - Ruhetag (im Kalender eingetragen) Art: Komplette Ruhe

Woche 2: 18.05. - 24.05.2026

Montag 18.05. - Ruhetag (fix) Art: Komplette Ruhe

Dienstag 19.05. - Trail-Prep Run Art: Easy Laufen mit Trail-Fokus Dauer: 60 min Herzfrequenz: Z2 (135-150 bpm) Pace: 5:00-5:30/km (bei Hügeln nur HR-Steuerung) Fokus: Start der ZUT Base-Phase, Trail-spezifische Bewegung

Mittwoch 20.05. - Easy Radfahren + Krafttraining Art: Easy Radfahren + Krafttraining Dauer: 75 min Rad + 45-60 min Kraft Herzfrequenz: Z1-Z2 (< 150 bpm) Fokus: Cross-Training + laufspezifische Stabilität

Donnerstag 21.05. - Hill Run (erste Trail-Einheit) Art: Trail/Hill Laufen Dauer: 50-60 min Herzfrequenz: Z2-Z3 (150-165 bpm, bei Anstiegen auf Körpergefühl hören) Pace: Nur HR-Steuerung, keine Pace-Vorgaben Fokus: Erste bergauf-bergab Gewöhnung für Zugspitz Ultratrail

Freitag 22.05. - Easy Run oder Ruhe Art: Easy Laufen oder aktive Recovery Dauer: 45 min (falls aktiv) Herzfrequenz: Z1 (< 135 bpm) Pace: > 5:30/km Fokus: Recovery nach erster Trail-Einheit

Samstag 23.05. - Long Run 3 (Trail-orientiert) Art: Long Run mit Trail-Elementen Dauer: 105-120 min Herzfrequenz: Z1-Z2 (135-150 bpm, bei Hügeln Z2-Z3) Pace: 5:00-5:30/km (flache Abschnitte), nur HR bei Hügeln Fokus: Längste Einheit der 2 Wochen, Trail-Ausdauer aufbauen

Sonntag 24.05. - Easy Radfahren Art: Easy Radfahren Dauer: 90-105 min Herzfrequenz: Z1-Z2 (< 150 bpm) Fokus: Aktive Recovery nach Long Run, Cross-Training

3) Wochenvolumen

Woche 1 (11.-17. Mai):

Gesamtstunden: 6-7.5h
Lauf-Kilometer: ~40-50 km
Rad-Kilometer: ~90-110 km
Woche 2 (18.-24. Mai):

Gesamtstunden: 7-8.5h
Lauf-Kilometer: ~50-60 km
Rad-Kilometer: ~100-120 km
Progression: Sanfter Übergang von Recovery (Woche 1) zu Trail-spezifischer Vorbereitung (Woche 2) gemäß Makroplan. Ab 19. Mai beginnt die ZUT Base-Phase für Zugspitz Ultratrail.
